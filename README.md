# Naughts and Crosses

This was created to try out the basics of react

## To Run:

 - clone this repository and navigate to it 
 ```
 git clone https://github.com/Le5tes/naughts-and-crosses-react.git
 cd naughts-and-crosses-react 
 ```
 - install dependencies
 ```
 npm install
 ```
 - run application
 ```
 npm start
 ```
